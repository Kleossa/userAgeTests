Сценарий тестирования (позитивный) возрастной группы Young:
1. Создать пользователя
2. Задать возраст 0
3. Проверить созданного пользователя (имя и возраст) // некорректое описание API
4. Проверить, что пользователь принадлежит к группе Young


Шаги выполнения сценария:
1. Заполнить имя в файле user.json
2. Выполнить файл positiveStep1.cmd
3. В ответе будет находиться идентификатор созданного пользователя
4. Заполнить возраст в файле age0.json и заполнить идентификатор, полученный на шаге 3
5. Выполнить файл positiveStep2.cmd
5. Выполнить файл positiveStep4.cmd
В результате выполнения positiveStep4.cmd в поле AgeGroup должно быть значение Young.
